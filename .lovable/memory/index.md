# Project Memory

## Core
DGTG NDIS Proactive Compliance Operating System. WCAG AAA+ accessibility required.
8 roles: super_admin, compliance_officer, supervisor, trainer, support_worker, hr_admin, executive, participant.
External Supabase for backend. Lovable AI for sentiment analysis & classification.
No record deletion — archive only. Full audit trail on every action.
Semantic design tokens only — never hardcode colors in components.
